¿Que es el internet de las cosas(IOT)?

Se denomina internet de las cosas a la interconexion digital entre dispositivos, personas y la propia internet que permiten el intercambio de datos entre ellos.

Un ejemplo sencillo es la conexión entre el smartphone y un dispositivo smart para controlar la iluminacion, entre otros.

¿Que podemos hacer con el IOT?

Con el iot podemos automatizar desde el riego de una planta hasta controlar el encendido y apagado de una lampara o artefacto electrico.

Como tambien se pueden recuperar distintos datos como humedad del suelo, consumo de agua, energia, entre muchas aplicaciones más.
